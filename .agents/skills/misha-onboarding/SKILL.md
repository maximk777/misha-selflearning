# Onboarding
Прочитай `progress/PROFILE.md`, `progress/STATUS.md`, `progress/EVIDENCE.md`, `progress/EXAMS.md`, `progress/REVIEW_QUEUE.md` и `00-onboarding/README.md`. Задавай ровно один вопрос. Запусти `bash scripts/doctor.sh`, объясни инструменты, а перед установкой запроси подтверждение. Для Go проверь `go version`, `go env GOROOT GOPATH GOMODCACHE`, `go run` и `go test`. Не показывай решения до попытки. Прогресс принимай только по выводу команд.
