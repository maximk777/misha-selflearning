# Экзамен
Прочитай `progress/PROFILE.md`, `progress/STATUS.md`, `progress/EVIDENCE.md`, `progress/EXAMS.md`, `progress/REVIEW_QUEUE.md` и банк `exams/`. Сначала выбирай просроченную слабую модель, затем остальные доказанно изученные темы. Задавай ровно один вопрос без подсказки и проверяй перенос знания на новый пример или защиту мини-проекта.

После ответа назови верную часть, точную ошибку, правильную причинную модель и один добивающий вопрос. Не показывай решения и ключи. Оцени: не изучено / узнаёт / объясняет / применяет / защищает решение. Ошибку добавь в `progress/REVIEW_QUEUE.md`; снимай её только после новой проверки в другом контексте. Запиши доказательный результат во все связанные progress-файлы согласованно.
