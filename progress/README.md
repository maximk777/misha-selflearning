# Прогресс

## Правила обновления

Файлы каталога хранят только подтверждённое состояние обучения. Не подменяй доказательства самооценкой и не отмечай лабораторию, тему или экзамен завершёнными без проверяемого результата.

- `PROFILE.md` — опыт, цели, ритм и тон общения;
- `STATUS.md` — текущий этап и короткий следующий шаг;
- `EVIDENCE.md` — журнал доказательств;
- `EXAMS.md` — история экзаменов;
- `REVIEW_QUEUE.md` — темы для интервального повторения.
