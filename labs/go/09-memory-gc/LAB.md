## Результат за 5–15 минут

```bash
cd labs/go/09-memory-gc/starter
go run .
go build -gcflags=-m .
```

Первая команда печатает `alloc before=… after=…`; вторая выводит решения escape analysis. Числа памяти между запусками не обязаны совпадать.
