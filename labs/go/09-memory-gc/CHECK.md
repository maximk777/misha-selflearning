# Проверка: memory и GC

> Банк: `exams/go-core.md`, GC-6

- [ ] Показан `go run .` output с двумя `Alloc` значениями.
- [ ] Показан `go build -gcflags=-m .` output.
- [ ] Ученик не делает вывод о скорости только из escape output.
- [ ] Ученик назвал ресурс, который GC не закроет.

Это advanced-проверка: её отсутствие не блокирует переход, но доказательство нужно для отметки темы.
