# Разбор 04: nil map

**Мисконцепция:** nil map «пустая, поэтому в неё можно писать» или zero value доказывает наличие ключа.

Чтение из nil map возвращает zero value, но запись требует allocate map: `make(map[string]int)`. `value, ok := scores[name]` разделяет существующий ключ со значением `0` и отсутствующий ключ.
