# Разборы лабораторий Go — только после попытки

Эта директория предназначена для экзаменатора и разбора уже показанной самостоятельной попытки. Не отправляй ссылки или текст решения ученику до прогноза, запуска, поломки и собственной починки. Каждый разбор называет именно ошибочную ментальную модель, а не только правильную строку кода.
