# Разбор 02: значения и указатели

**Мисконцепция:** «передача в функцию всегда меняет исходную переменную» или «указатель передаётся не по значению».

`IncrementValue` получает copy `Counter`; изменение copy не меняет caller. `IncrementPointer` тоже получает копию argument, но эта копия содержит address исходного объекта, поэтому `counter.N++` меняет объект по адресу. Pointer выбирают по семантике mutation/ownership, не по догме о скорости.
