# Разбор 09: memory и GC

**Мисконцепция:** любой pointer обязательно heap allocation, а `runtime.GC()` делает память постоянной и одинаковой.

Escape analysis — решение компилятора о времени жизни, не моральная оценка кода. `MemStats.Alloc` зависит от runtime и окружения; учебный вывод показывает способ наблюдать, но не служит benchmark. GC освобождает недостижимую память, но не закрывает external resources.
