# Разбор 10: pprof

**Мисконцепция:** наличие pprof endpoint автоматически находит проблему или profile без нагрузки достаточен.

Blank import `net/http/pprof` регистрирует handlers. `/work` создаёт повторяемое действие, которое можно наблюдать; без зафиксированного workload сравнение profiles бессмысленно. Сервер запускается только локально и останавливается после опыта.
