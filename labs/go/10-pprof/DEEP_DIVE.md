# Deep dive: CPU profile по воспроизводимому действию

> Уровень: `advanced`
> Время: 30 минут
> Не блокирует minimum path.

Запусти сервер, затем в другом terminal выполни несколько `curl http://127.0.0.1:6060/work` и `go tool pprof http://127.0.0.1:6060/debug/pprof/profile?seconds=2`. Останови сервер Ctrl-C после снимка.

**Один вопрос:** почему profile после нулевой нагрузки нельзя сравнивать с profile под нагрузкой? Сначала закрепи одно действие, интенсивность и длительность, затем сравни одну гипотезу.
