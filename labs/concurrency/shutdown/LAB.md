# Graceful shutdown

Предскажи, будут ли уже принятые jobs потеряны. Запусти `go test ./shutdown/starter`. Поломка: временно удали `wg.Wait()` перед return и запусти тест несколько раз; ожидай потерянную работу. Восстанови drain, затем перескажи порядок: остановить intake → дождаться принятых jobs → вернуть результат.
