# Установка Go
Определи ОС через `uname -s` и архитектуру через `uname -m`. Выбери официальный installer с go.dev/dl или системный package manager и запроси подтверждение. Проверь `go version` и `go env GOROOT GOPATH GOMODCACHE`. Создай модуль через `go mod init`, выполни `go run .` и `go test ./...`, затем объясни роль каждого каталога.
