## Модель

Интерфейс в Go задаёт набор методов, а тип удовлетворяет ему неявно. Маленький interface обычно принадлежит consumer, которому нужно поведение. Композиция строит тип из частей без наследования и без обязательной иерархии.

## Прогноз → опыт → поломка
