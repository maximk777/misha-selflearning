# Модуль 10: итоговый Task API

Финальный проект собирает уже изученные блоки: HTTP, Go domain, PostgreSQL, Redis, Kafka/outbox, context и containerization. Быстрый minimum path запускается без Docker: `cd project/backend-lab && go test ./...`.

Каждый checkpoint требует один прогноз, запуск, безопасную поломку, самостоятельную починку и пересказ. Прогресс фиксируется только по тесту, выводу, коду или принятой защите.
