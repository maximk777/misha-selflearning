# Races, deadlocks, leaks

Race — несинхронизированный одновременный доступ, где есть запись: ищи `go test -race ./...`. Deadlock — все ждут: запускай только через `timeout`/контекст и исправляй порядок lock. Leak — работа не узнаёт об отмене: добавь `ctx.Done()` и дождись worker. Не лечи проблему `Sleep`.
