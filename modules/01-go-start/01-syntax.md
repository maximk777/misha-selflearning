# Синтаксис и первый запуск

> Уровень: `start`
> Время: 45 минут
> Предпосылки: Go установлен
> Практика: `labs/go/01-syntax/LAB.md`
> Экзамен: `exams/go-start.md`, GS-1

## Результат за 5–15 минут

Выполни `cd labs/go/01-syntax/starter && go run .`: программа дважды напечатает приветствие с номером запуска.

## Модель

Go компилирует исходный код заранее. Переменная получает конкретный тип, а блоки `if` и `for` задают обычный поток выполнения. Короткая запись `:=` доступна только там, где появляется хотя бы новая переменная.

## Прогноз → опыт → поломка

**Один вопрос:** сколько строк будет в выводе и почему? Запусти программу. Затем замени `attempt <= 2` на `attempt < 2`, зафиксируй изменение и верни исходное поведение.
