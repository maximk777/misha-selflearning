# Errors, defer, panic и recover

> Уровень: `interview`
> Время: 90 минут
> Предпосылки: functions, maps
> Практика: `labs/go/06-errors-panic/LAB.md`
> Экзамен: `exams/go-start.md`, GS-5

## Результат за 5–15 минут

Выполни test лаборатории: `FindUser` возвращает error, который распознаётся через `errors.Is`, а `MustPositive(0)` панически завершает только явно неверное использование.

## Модель

`error` — обычное значение, которое caller обязан рассмотреть. `defer` выполняется при выходе из функции и удобен для cleanup. `panic` — не замена ожидаемой ошибки; `recover` уместен на редкой границе процесса/worker, чтобы вернуть контролируемую ошибку или лог.
