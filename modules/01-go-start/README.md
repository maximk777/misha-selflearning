# Модуль 01: Go Start

Цель модуля — за две недели перейти от первого запуска Go до уверенного чтения простого кода: значения, ссылки на данные, ошибки и границы `panic`. Минимальный путь — темы 01–09 и лаборатории `labs/go/01-syntax` … `labs/go/06-errors-panic`; глубже в рантайм пока не уходим.

Каждая тема начинается с прогноза, затем запускается маленький опыт. Не открывай `labs/go/solutions/` до самостоятельной попытки: это материал экзаменатора для разбора уже сделанной работы.

| Тема | Быстрый результат | Практика | Банк |
|---|---|---|---|
| 01–03 | модуль, package, функция | `labs/go/01-syntax` | GS-1 |
| 04–06 | bytes/runes, slices, maps | `labs/go/03-arrays-strings-slices`, `labs/go/04-maps` | GS-2–GS-4 |
| 07–08 | struct, method, value/pointer | `labs/go/05-structs-methods`, `labs/go/02-values-pointers` | GS-6 |
| 09 | error, `defer`, `panic` | `labs/go/06-errors-panic` | GS-5 |
