# Версии инструментов

| Инструмент | Версия курса | Назначение |
| --- | --- | --- |
| Go | 1.26.5 | Язык и стандартные инструменты лабораторий |

Версия Go сверена с официальной страницей релизов 21 июля 2026 года. Корневой модуль валидатора сохраняет директиву `go 1.24.0`, чтобы стартовая структурная проверка запускалась на уже установленном bootstrap-инструменте; учебные лаборатории будут закреплять версию курса в своих инструкциях и модулях.

Версии PostgreSQL, Redis, Kafka и Docker-образов будут добавлены вместе с первыми лабораториями, которые их используют, чтобы не создавать фиктивные зависимости.
